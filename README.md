</ul>
<p> The Functionalities of the projects are listed below.</p>
<h1>TodoList functions</h1>
<li>Addtask: The User can add a task.</li>
<li>Complete Task:  User can Checkout task that has been completed.</li>
<li>Delete Task: User can delete Task that was not completed of failed to achieve.</li>
<h2>Weekly progress report: <small> User can view their weekly or monthly progress which includes the following.</small></h2>
<li>Task deleted:Which includes task that was deleted.</li>
<li>Task Completed: Which includes task that was completed or checked.</li>
<li>Task Added: Which includes task that was  added in the input box.</li>
<li>Total tracked task: Which includes all the task added.</li>
<li>Still pending task: Which includes task that has  been added but has not been completed neither deleted.</li>
